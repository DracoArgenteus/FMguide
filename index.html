Folkemødet 2025 - Interactive Event Schedule

Browse, search, and sort events. Filter by weekday. Click on descriptions or locations for more info.

How to Use This Page

  1. Ensure `folkemoedet_events.csv` is in the same folder as this HTML file.
  2. Use the buttons to filter by weekday.
  3. Click on any event description that ends with "..." to see the full details in a pop-up.
  4. Click on a location to see it on the official Folkemødet program map (opens in a new tab).
